# Frontend for BioFlux

This React + Vite frontend provides a role-based dashboard experience for the BioFlux application.

## Install dependencies

From the `frontend` folder:

```powershell
npm install
```

## Run locally

```powershell
npm run dev
```

Open `http://localhost:4173` in your browser.

## Build for production

```powershell
npm run build
```

The production output will be in `frontend/dist`.

## Backend integration

When the frontend is built, the backend can serve the static files from `frontend/dist`.
